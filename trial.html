Airbnb Mobile App Homepage Clone

This project is a frontend clone of the Airbnb mobile app homepage, built using HTML and CSS. The goal was to replicate the look and feel of Airbnb’s mobile interface, including the navigation bar, search bar, category buttons, listings, and footer icons

Airbnb Mobile App Homepage Clone

  • Responsive mobile layout with a simulated phone screen
  • Search bar and category buttons for filtering listings
  • Property listings with images, location, and pricing details
  • Sticky navigation and footer for easy app-like interaction

This project was created to practice frontend development skills, improve CSS layout techniques, and better understand UI/UX design principles in modern web applications.
